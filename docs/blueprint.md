# **App Name**: HTML SnippetVault

## Core Features:

- Snippet Input Form: Users can input new HTML code snippets along with a descriptive title or tag, allowing for organized storage.
- AI-Powered Title Suggestion: An AI tool analyzes uploaded HTML code to suggest concise, descriptive titles automatically, streamlining the organization process.
- Search Functionality: A dynamic search bar enables users to filter and locate specific HTML snippets based on keywords in their descriptive titles.
- Snippet List Display: A clear, browsable list of all stored snippet titles is displayed, which updates in real-time with search results.
- Code Viewer: Upon selecting a snippet from the list, its full HTML code content is revealed and displayed in a dedicated, scrollable viewing panel.
- One-Click Copy: A convenient 'Copy' button allows users to quickly copy the currently displayed HTML code of a selected snippet to their clipboard.
- Persistent Storage: All HTML snippets and their associated descriptive titles are securely stored and retrieved, ensuring data persistence across user sessions.

## Style Guidelines:

- Primary color: A deep, professional indigo blue (#335599) selected to convey a technical and trustworthy feel suitable for a development tool.
- Background color: A very light, desaturated cool gray (#ECF0F3) serving as a clean, calming canvas that enhances code readability and reduces eye strain.
- Accent color: A vibrant yet functional cyan (#2EBEDB) to highlight interactive elements, calls-to-action, and key information, ensuring critical actions stand out.
- Body and headline font: 'Inter' (sans-serif) chosen for its modern, neutral, and objective appearance, offering high readability across various screen sizes. Code snippets: 'Source Code Pro' (monospace) for optimal clarity and consistency in code presentation.
- Utilize a set of minimalist line icons for core functionalities such as 'copy', 'add', and 'search', aligning with a clean and professional user experience.
- A responsive three-panel layout featuring a sidebar for navigation and filters, a central area for displaying the snippet list, and a dedicated detail panel for selected code, maximizing productivity and content visibility.
- Implement subtle hover effects on interactive elements and smooth, brief transitions when revealing or expanding code snippets, providing refined feedback without distracting the user.
